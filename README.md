## MVP
> En recept-app för att hitta recept baserat på vilka ingredienser man har hemma, med möjlighet att spara favoritrecept och sökfilter.

- [ ] Sökfält med multipla sökord samt olika filter.
- [ ] Överskådlig vy av sökresultat i "card"-format.

- [ ] Funktion för att spara favoritrecept (tillgängligt på flera ställen)
- [ ] Välja ett recept och få upp i helskärmsläge.
- [ ] Lätt att läsa recept och kunna bocka av steg-för-steg.

- [ ] Möjlighet att skapa användarprofil
- [ ] visa sparade recept
- [ ] sparade sökfilter
- [ ] inställningar
